# KI-Kostenrechner

Vergleiche die API-Kosten der wichtigsten KI-Modelle (GPT-4o, Claude, Gemini) interaktiv.

**Live:** https://jonas-keil.github.io/ki-kostenrechner

## Features
- Live-Kostenvergleich fuer 6 KI-Modelle
- Slider fuer Anfragen, Woerter pro Anfrage & Antwort
- Guenstigstes Modell automatisch hervorgehoben
- Responsive Design (Mobile-first)
- Preise in `data/modelle.json` einfach aktualisierbar
