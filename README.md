# KI-Kostenrechner

Vergleiche die API-Kosten der wichtigsten KI-Modelle (GPT-5.4, Claude, Gemini) interaktiv – inklusive Intelligence Score von Artificial Analysis.

**Live:** https://jonas-keil.github.io/ki-kostenrechner

## Enthaltene Modelle (Stand: März 2026)

| Anbieter | Modell | Intelligence Score |
|----------|--------|--------------------|
| OpenAI | GPT-5.4 | 57 |
| OpenAI | GPT-5.4 Pro | – |
| Anthropic | Claude Opus 4.6 | 53 |
| Anthropic | Claude Sonnet 4.6 | 52 |
| Anthropic | Claude Haiku 4.5 | 37 |
| Google | Gemini 3.1 Pro | 57 |
| Google | Gemini 3.1 Flash-Lite | 34 |

*Intelligence Scores: [Artificial Analysis Intelligence Index](https://artificialanalysis.ai/leaderboards/models)*

## Features
- Live-Kostenvergleich fuer 8 KI-Modelle
- Intelligence Score (Artificial Analysis) je Modell
- Slider fuer Anfragen, Woerter pro Anfrage & Antwort
- Guenstigstes Modell automatisch hervorgehoben
- Responsive Design (Mobile-first)
- Preise und Scores in `data/modelle.json` einfach aktualisierbar
